Diagnostics

The UltraESB allows one to diagnostic each ESB node in multiple ways:

  • Log Viewer - View the tail end of the log file through the UConsole
  • Loggers - Manage the available loggers and change the log levels and additivity
  • Appenders - Manage available appenders and change the log pattern or threshold
