Monitoring

The UltraESB allows one to monitor each ESB node in multiple ways as follows:

  • Metrics Dashboard - View a set of different style graphs to monitor artifacts and the node
  • Alerts Dashboard - View the system generated alerts and configure what triggers an alert and its behavior
  • Zabbix Registration - Register monitorable elements of this instance against a Zabbix server
  • Zabbix Template Manager - Manage templates used to register monitorable elements on Zabbix
