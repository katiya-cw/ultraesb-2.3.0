Node Management

Facilitates the management of a single node of the UltraESB. On a cluster of nodes, this represents the currently connected node. To manage another node, one must switch the connected instance using the link at the top right hand corner.

Node management allows one to manage the many aspects of an UltraESB node instance - such as the configuration, Proxy services, Sequences, Endpoints, File caches, Work Managers, and Transports. Navigate to the appropriate category for more information
